## ℹ️앱 설명

음악을 재생, 정지 및 종료할 수 있는 기능을 가진 음악 플레이어 앱

<br>

## ✅구현 기능

* MediaPlayer를 이용한 음원 재생
* Service를 이용한 백그라운드 음원 재생
* Notification에 음원 컨트롤러 제공

<br>

## ✅사용되는 기능

* Android
  * MediaPlayer
  * Service
  * Notification
    * PendingIntent
    * Intent flag
